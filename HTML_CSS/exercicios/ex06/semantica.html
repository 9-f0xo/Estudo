Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. (não semântica)
Nessa frase temos um termo em destaque usando a tag <strong>. (semântico)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i>. (não semântica)
Nessa frase temos um termo em ênfase usando a tag <em>. (semântico)

Texto Marcado

Nessa frase temos um termo marcado usando a tag <mark>. (não semântica)

Texto Grande e Pequeno

Nessa frase temos um texto grande usando a tag <big>. (não semântica, fora de uso)
Nessa frase temos um texto pequeno usando a tag <small>. (semântica)

Texto Deletado e Inserido

Marcamos um texto 'Deletado' usando a tag <del>, usado quando queremos indicar que ele deve ser lido, mas é não considerado. (semântica)
Marcamos um texto 'Inserido' usando a tag <ins>, usado quando queremos indicar que ele foi adicionado depois. (semântica)
(Existe a tag <u> para sublinhar, mas é não semântica)

Texto Sobrescrito e Subscrito

Para coisas do tipo x2+4=8 usamos a tag <sup>
Para coisas do tipo H2O usamos a tag <sub>
